# 🎓 SkillSphere

Site de plataforma de cursos online desenvolvido como projeto prático durante um curso da **Alura**. Página inicial com header, hero, seção de conteúdo e footer.

## 📄 O que tem na página

- **Header** fixo com logo, barra de busca e botões de login/cadastro
- **Hero** com título, subtítulo e botão de call-to-action ("Ver Planos")
- **Seção de conteúdo** textual
- **Footer** com créditos e link do desenvolvedor

## 🛠️ Tecnologias

- HTML5 / CSS3
- JavaScript (vanilla)
- Deploy via GitHub Pages

## 📁 Estrutura

```
SkillSphere/
├── index.html    # Estrutura da página
├── style.css     # Estilização
├── main.js       # Scripts
└── assets/       # Logo, ícone e imagens
```

## 📌 Contexto

Projeto de curso (Alura) focado em praticar estruturação de layout, organização de header com múltiplos elementos e composição de hero section. O conteúdo da página é placeholder — o foco estava na estrutura e no CSS.
